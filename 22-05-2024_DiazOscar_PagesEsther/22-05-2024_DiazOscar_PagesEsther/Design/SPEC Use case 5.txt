Use case Number: 5
Use case Name: CreateNewSpreadsheet

Actors and their interests:
System

Preconditions:
- There system does not have any spreadsheet

Postconditions:
- The system creates a spreadsheet ready for content insertion or modification

Main Success Scenario (Basic Flow):
1. The system initiates the process of creating a new spreadsheet
2. The system prepares an empty data structure to represent the cells
3. The system displays the new spreadsheet

Extension (Alternative Flow):
1.a The system notifies an error creating the spreadsheet
	1. The system notifies the user

SPECIFICATIONS:
void execute(Spreadsheet spreadsheet, S2VLoader loader, S2VStore store)
- Belongs to CreateSpreadsheet class and is a command pattern, responsible for creating and saving a new empty spreadsheet.
- Prompts the user for a filename to save the new spreadsheet. Creates a new Spreadsheet instance. Creates a new S2VStore instance with the filename and the newly created Spreadsheet, then calls storeSpreadsheet to save it. Handles SavingSpreadSheetException in case of errors during saving. Displays the newly created empty spreadsheet using display().
